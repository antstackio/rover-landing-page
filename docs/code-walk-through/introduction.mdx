# Introduction


Rover comprises of two packages
 
 * rover-cli 
 * rover-engine

### Rover CLI
Rover CLI is the main package which imports rover-engine. this package accepts input from CLI and convert it into a JSON, this json will given as input to respective function in rover-engine and the project will be generated.

### Rover Engine
Rover Engine is the package which comprises of functions which creates,updates or deploys SAM projects.
it also comprises of function which helps CLI in SAM validation, listing resources and etc.
