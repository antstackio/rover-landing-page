#  Bug Fix


You can contribute to rover by working on the issues listed here [GitHub Page](https://github.com/antstackio/rover-cli/issues) or If you find a issue report it by  following [Bug Report](bug-report-or-feature-request)



### Bug fix

1. Set Up rover in your system by following [Rover Set Up](rover-set-up).

2. Inspect the issue 

3. On resolving it create a PR as per the format.
