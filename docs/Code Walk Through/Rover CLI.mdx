# Rover CLI


Rover comprises of two packages one is rover-cli and other one is rover-engine, where rover-cli takes user input converts it to a rover required config  , and then using the config and engine it generates the project.

Rover CLI
Rover CLI is the main package which imports rover-engine. this package accepts input from CLI and convert it into a JSON, this json will given as input to respective function in rover-engine and the project will be generated.
